# Changelog

## 0.1.0

- Added `safecopy plan`, `safecopy pack`, and `safecopy inspect`.
- Added built-in deny globs for common secret, cache, dependency, and generated paths.
- Added configurable `safecopy.config.json` includes, denies, max file bytes, and redaction rules.
- Added deterministic manifests with copied files, skipped files, hashes, and redaction counts.
- Added fixture-backed tests, smoke script, validation script, and CI workflow.
